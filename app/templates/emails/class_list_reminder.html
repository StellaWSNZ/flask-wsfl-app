Key steps

  1. Go to Upload Class List
  2. Select your school, term, and year
  3. Enter the teacher name and class name
  4. Upload your file (.csv, .xls, .xlsx, or .xlsm)
  5. Match your columns to the correct WSFL field names
  6. Validate the data and fix any critical errors
  7. Submit the class once validation is complete

Required fields

  • NSN
  • LastName
  • either FirstName or Preferred Name
  • Year Level (if not already in database)

Helpful notes

  • Leave unnecessary columns blank when assigning fields
  • Red cells are critical errors and must be fixed before submitting
  • Yellow cells are warnings and may still be submitted, but can overwrite current data
